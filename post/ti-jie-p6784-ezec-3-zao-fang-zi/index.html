题解 P6784 「EZEC-3」造房子

2020-11-15 题解

题意：

有 ccc 块钱、aaa 个 AAA 材料和 bbb 个 BBB 材料，造第 iii 层楼需要 iii 个 AAA 材料与 iii 个 BBB 材料，每块钱都可以用来买 111 个 AAA 材料或者 111 个 BBB 材料，问最多可以造几层楼。

思路：

要求最多可以造几层楼，那么只要暴力枚举第 iii 层即可。

注意数据范围，要开 ullullull

Code:

#include <bits/stdc++.h>
using namespace std;
typedef unsigned long long ull;
ull a,b,c;
ull ans;
int main() {
	cin>>a>>b>>c; //输入 
	if(a>b) swap(a,b); //如果 a>b 就交换 a,b 的值 
	for (ans=1;;ans++) { //无条件循环枚举每一层 
		if(a<ans) { //如果 a<ans 
			if(a+c >= ans) {
				c-=ans-a;
				a=ans;
			}
			else break;
		}
		if(b<ans) { //如果 b<ans 
			if(b+c >= ans) {
				c-=ans-b;
			    b = ans;
			}
			else break;
		}
		a-=ans, b-=ans;
	}
	cout<<ans-1; //注意输出的是 ans-1 
	return 0;
}
