题解 CF994A 【Fingerprints】

发布于 2020-11-15 | 分类于 题解 | 1分钟 | 165字数

题意：

按顺序输出第一个数组中在第二个数组中的数。

这题两个数组都不会出现重复数字。

思路：

输入两个数组，二重循环遍历 xxx 数组，如果 xi=yix_i=y_ixi​=yi​，就输出 xix_ixi​。

Code:

#include <bits/stdc++.h>
using namespace std;
int n,m;
int x[15],y[15];
int main() {
	cin>>n>>m;
	for (int i=1; i<=n; i++) cin>>x[i];
	for (int i=1; i<=m; i++) cin>>y[i]; //读入两个数组
	for (int i=1; i<=n; i++) {
		for (int j=1; j<=m; j++) {
			if(x[i] == y[j]) cout << x[i] << " "; //如果找到相等的就输出
		}
	}
	return 0;
}
